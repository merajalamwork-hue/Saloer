## 🎯 Scope of Testing

The test cases cover the complete end-to-end shopping flow:

| Area                     | Description                                          |
|--------------------------|------------------------------------------------------|
| 🏠 Homepage & Navigation  | Page load, category navigation, search               |
| 📋 Product Listing (PLP)  | Product cards, sorting, filtering                    |
| 🔍 Product Detail (PDP)   | Images, variants, add to cart                        |
| 🛒 Shopping Cart          | Add/remove items, quantity update, cart persistence  |
| 📧 Checkout — Contact     | Guest email entry, validation                        |
| 📦 Checkout — Shipping    | Address form, required fields, shipping methods      |
| 💳 Checkout — Payment     | Valid/invalid card scenarios                         |
| ✅ Order Confirmation      | Confirmation page, email notification                |
| ⚠️ Edge Cases             | Out-of-stock, empty cart, mobile responsiveness      |
